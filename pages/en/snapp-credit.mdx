# Приложение Mina SNAPP Credit 

Приложение Mina SNAPP Credit уже доступно! С приложением Mina SNAPP Credit вы сможете создать доказательство SNARK, которое подтверджает [Teller Finance](https://www.teller.finance/), что ваш кредитный рейтинг превышает кредитный порог в размере 700. 

Было разработано приложение [Electron](https://www.electronjs.org/) для Ubuntu, macOS и даже WSL2. Чтобы начать использование приложения, убедитесь, что у вас есть подтвержденный [аккаунт Credit Karma](https://www.creditkarma.com/) (поддерживаются оба домена .COM и .CA). Это приложение будет проверять только аккаунты Credit Karma, поддержка для других кредитных организаций в данный момент отсутствует. 

## Установка приложения Mina SNAPP Credit 

Первым шагом к созданию кредитного SNARK-доказательства является установка приложения Mina SNAPP Credit. Следуйте приведенным ниже инструкциям для вашей операционной системы. 

### macOS

Перед установкой на Mac вам необходимо установить некоторые зависимости с помощью [Homebrew](https://brew.sh/). Следуйте инструкциям ниже, чтобы загрузить все необходимые зависимости: 

1. Если у вас не установлен `Homebrew`, вы можете установить его, выполнив следующую команду в вашем терминале: 

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

2. После того, как вы установили `Homebrew`, запустите следующую команду в своем терминале для загрузки необходимых зависимостей: 

```
brew install jemalloc gmp libsodium openssl@1.1
```

3. Загрузите версию для macOS [здесь](https://storage.googleapis.com/mina-snapp-credit-app-packages/snapp-credit-check-mac.dmg).

### Ubuntu 20.04

Загрузите версию Ubuntu 20.04 [здесь](https://storage.googleapis.com/mina-snapp-credit-app-packages/snapp-credit-check-ubuntu20.04.deb). После загрузки вы можете ее установить, выполнив следующую команду в своем терминале: 

```
sudo add-apt-repository universe
sudo apt update
sudo apt install -y ~/Downloads/snapp-credit-check-ubuntu20.04.deb
```

### Ubuntu 18.04

Загрузите версию Ubuntu 18.04 [здесь](https://storage.googleapis.com/mina-snapp-credit-app-packages/snapp-credit-check-ubuntu18.04.deb). После загрузки вы можете ее установить, выполнив следующую команду в своем терминале: 

```
sudo add-apt-repository universe
sudo apt update
sudo apt install -y ~/Downloads/snapp-credit-check-ubuntu18.04.deb
```

### WSL

Можно запустить приложение WSL2, но вам нужно будет настроить пересылку X11, чтобы посмотреть, как приложение работает на вашем локальном компьютере. Смотрите это [руководство](https://stackoverflow.com/questions/61110603/how-to-set-up-working-x11-forwarding-on-wsl2), чтобы настроить пересылку X11 в вашей системе.
Если на вашем компьютере уже настроена пересылка X11 на Windows, вы можете использовать [версию Ubuntu 20.04](https://storage.googleapis.com/mina-snapp-credit-app-packages/snapp-credit-check-ubuntu20.04.deb), чтобы начать работу. 
Для установки выполните следующие команды в терминале WSL2: 

```
sudo apt update
sudo apt install -y /path/to/snapp-credit-check-ubuntu20.04.deb
```

## Отправка Teller

Чтобы начать отправку SNARK-доказательства и начать работу, посетите [сайт Teller](https://mina.teller.finance/). Вам понадобится кошелек, подключенный к вашему браузеру, мы рекомендуем [Metamask](https://metamask.io/). Кроме этого, убедитесь, что вы подключились к [Rinkeby Testnet](https://www.rinkeby.io/). Вам понадобятся токены ETH для оформления ссуды, поэтому обязательно откройте [кран Rinkeby](https://faucet.rinkeby.io/) и запросите несколько токенов. И последнее примечание: пожалуйста, не используйте аппаратный реестр.

После того, как вы зайдете на сайт Teller, выберите кошелек для подключения. Убедитесь, что вы вошли в свой кошелек, чтобы вы смогли подключиться к Teller.

## FAQ

1. Я получаю сообщение `Login unsuccessful. Please check your credentials or restart the application and try again`, что это означает? 

Обычно это означает, что учетные данные Credit Karma неверны, и приложение не может подключиться к Credit Karma. Проверьте свои учетные данные и попробуйте еще раз. 

Если ваши учетные данные верны и вы по-прежнему получаете это сообщение об ошибке, скорей всего, приложение Mina не может открыть экземпляр Credit Karma. В этом случае попробуйте закрыть приложение и удалить все запущенные процессы приложения Mina. 

2. Я получаю сообщение `Credit score is less than 700, cannot produce SNAPP proof to: /.../snapp-credit-score`, что это означает? 

Это означает, что кредитный рейтинг пользователя ниже порогового диапазона 700, и поэтому доказательство не может быть сгенерировано. 

3. Я получаю сообщение `Failed to generate proof to: /.../snapp-credit-score`, что это означает? 

Эта ошибка указывает на то, что приложению Mina не удалось создать доказательство SNARK. 

Если возникает эта ошибка, создайте запрос на решение этой [проблемы](https://github.com/MinaProtocol/mina/issues) в Github Mina, указав заголовок, включающий `SNAPP Credit App`, а также информацию о вашей операционной системе. 

4. Нужна ли мне Mina для создания SNAPP-доказательства для Teller? 

Нет, вам не нужна Mina для создания SNAPP-доказательства. 

Мы создали сеть под названием `Snappnet` таким образом, что несколько определенных аккаунтов оплачивают все сборы. Чтобы создать SNAPP-доказательство, вам нужен лишь собственный адрес Ethereum и актуальные учетные данные Credit Karma. 

5. Действительны ли представленные доказательства? 

Сеть Mina `Snappnet` не является основной сетью Mina, так как она работает наподобие тестовой сети. Teller Finance использует [Rinkeby Testnet](https://www.rinkeby.io/) для выдачи займов со своей стороны, и это не зависит от текущей сети. 

6. Когда я смогу участвовать в `Snappnet`?

В настоящее время только Teller подключен к `Snappnet`. Поздже мы включим протоколы SNAPP, чтобы все могли подключиться и принимать участие.

7. Подтверждение моего SNAPP-доказательства в Teller занимает много времени, в чем проблема? 

Если вы заметили, что проверка вашего доказательства занимает больше 20 минут, опубликуйте сообщение на канале Mina Discord. 

8. Metamask работает некорректно, в чем проблема? 

Убедитесь, что вы подключились к [Rinkeby Testnet](https://www.rinkeby.io/) в своем кошельке Metamask. 

9. Мне нужен ETH для оплаты газа, чтобы оформить заявку на кредит, как мне получить средства? 

Пожалуйста, откройте [кран Rinkeby](https://faucet.rinkeby.io/)  запросите средства на указанный вами кошелек Metamask.
